ReportCard (Bulletin)

Rôle métier

Représente le bulletin officiel d’un élève pour une période donnée, dans une année scolaire précise.

Le bulletin est une vue de synthèse figée, calculée à partir des StudentAssessment VALIDATED`.

⚠️ Aucune note n’est saisie ici manuellement.


---

Dépendance hiérarchique (très important)

StudentAssessment
   ↓ (agrégation)
ReportCard

Le bulletin n’existe que si :

les évaluations sont CLOSED

les notes sont VALIDATED

Contraintes

Unicité

(student_enrollment_id, school_year_cycle_term_id) UNIQUE


Cohérence

L’élève doit être ACTIF durant la période

Toutes les Assessment du cycle doivent être CLOSED



---

Logiques métier

Moyennes calculées via :

normalized_score

coefficients (SchoolYearLevelSubject)


Les absences non justifiées pénalisent la moyenne (règle paramétrable)

Le rang est calculé par classe



---

Règles de gel

Une fois LOCKED :

aucune modification possible

toute correction passe par annulation + nouveau bulletin

Ce qu’on ne met PAS ici
